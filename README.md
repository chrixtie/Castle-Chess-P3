# Castle Chess Tournament Management App

This is a console-based Python application designed to manage chess tournaments
offline.

## Features
- Player management
- Tournament rounds and match results
- Automatic score calculation
- JSON-based data storage
- Offline use

## Installation
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies using `pip install -r requirements.txt`

## Running the App
bash
python main.py
